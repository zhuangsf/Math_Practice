# AGENTS.md

## 基本要求
1，先做好设计再构建代码框架最后执行开发
2，代码要求编码严谨，考虑充分，做好解耦和重构。
3，添加必要的注释和log，特别时较复杂的逻辑。
4，代码修改完成后，需要执行单元测试测试例，确保逻辑正确代码可靠。

## 指令要求
1，当要求提交代码时执行
先更新代码结构，更新README.md文档。之后执行
git add .
git status --short
生成commit文件commit_msg.txt
git commit -F commit_msg.txt
git push origin main
git log --oneline -1

 For more details see `README.md` in the repository root.
